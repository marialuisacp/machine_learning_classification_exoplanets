Preparação e leitura dos dados¶

Neste trabalho estamos interessados em resolver um problema de classificao binria de candidatos a exoplanetas encontrados pela sonda espacial Kepler da NASA. Um exoplaneta um planeta fora do sistema solar (i.e. que no orbita o sol). A sonda primeiro identifica sinais de possveis exoplanetas, chamados de Kepler Object of Interest (KOI). Porm nem todos os KOIs so de fato exoplanetas, alguns se tratam de falsos positivos de origens diversas. A tarefa ento classificar os KOIs entre exoplanetas confirmados e falsos positivos. Cada linha do arquivo corresponde a um KOI e as features so caractersticas estimadas de cada (possvel) exoplaneta (tamanho, temperatura, features da estrela hospedeira, etc).

Dados retirados do NASA Exoplanet Archive.

    plt.title(nameParam + ' x Acurcia')

Decision Tree¶

rvores de deciso so mtodos de aprendizado de mquinas supervisionado no-paramtricos, muito utilizados em tarefas de classificao e regresso. Este mtodo representa funes como rvores de deciso. Estas rvores so treinadas de acordo com um conjunto de treino (exemplos previamente classificados) e posteriormente, outros exemplos so classificados de acordo com essa mesma rvore.

Support Vector Machine (SVM)¶

Uma mquina de vetores de suporte um mtodo de aprendizado supervisionado que analisa os dados e reconhece padres, usado para classificao e anlise de regresso. O SVM toma como entrada um conjunto de dados e prediz, para cada entrada dada, qual de duas possveis classes a entrada faz parte, o que faz do SVM um classificador linear binrio no probabilstico.

Um modelo SVM uma representao de exemplos como pontos no espao, mapeados de maneira que os exemplos de cada categoria sejam divididos por um espao claro que seja to amplo quanto possvel. Os novos exemplos so ento mapeados no mesmo espao e preditos como pertencentes a uma categoria baseados em qual o lado do espao eles so colocados.

In [19]:
#
# Funcao que treina o modelo e mostra os resultados utilizando o SVM
#

def test_svm(c, kernel, gamma):
    clf = svm.SVC(C=c, kernel=kernel, gamma=gamma)

    clf = clf.fit(x_train, y_train)

    train_acc = clf.score(x_train, y_train)
    val_acc = clf.score(x_test, y_test)

    print("Acuracia do treino: {:.3}".format(train_acc))
    print("Acuracia do teste: {:.3}".format(val_acc))

O SVM recebe um parmetro C, que um parmetro de otimizao. Ele informa otimizao do SVM quanto se deseja evitar classificar erroneamente cada exemplo de treinamento. Para valores grandes de C, a otimizao escolher um hiperplano de margem menor se esse hiperplano fizer um trabalho melhor ao classificar todos os pontos de treinamento corretamente. O valor muito pequeno de C faz com que o otimizador procure por um hiperplano de separao de margem maior, mesmo se esse hiperplano classificar incorretamente mais pontos.

Kernels¶

O SVM introduz a funo kernel que transforma o espao de dados original em um novo espao com dimenses mais altas. Existem vrias frmulas para a funo kernel, neste trabalho estamos trabalhando com o kernel linear, sigmoide, polinomial e radial.

Linear¶

til ao lidar com grandes vetores de dados esparsos. frequentemente usado na categorizao de texto. O kernel linear tambm funciona bem em problemas de regresso. A equao dada por:

$$ K(X,Y)=X^TY $$

Polinomial¶

Este kernel induz o espao de combinaes polinomiais at certo ponto. Consequentemente, pode-se trabalhar com limites de deciso ligeiramente "dobrados", como parbolas com grau = 2. Em alguns casos combinaes polinomiais no so flexveis o suficiente para "dobrar" o espao com fora suficiente para capturar o ponto de interesse. Este kernel popular no processamento de imagens. A equao dada por:

$$ K(X,Y)=(X^TY+r)d,>0 $$

k-Nearest Neighbors (KNN)¶

K-Nearest Neighbors um dos muitos algoritmos de aprendizagem supervisionada usado no campo de data mining e machine learning, ele um classificador onde o aprendizado baseado no quo similar um dado (um vetor) do outro. O treinamento formado por vetores de n dimenses.

In [24]:
#
# Funcao que treina o modelo e mostra os resultados utilizando o KNN
#

def test_knn(k):
    clf = neighbors.KNeighborsClassifier(n_neighbors=k)

    clf = clf.fit(x_train, y_train)

    train_acc = clf.score(x_train, y_train)
    val_acc = clf.score(x_test, y_test)

    print("K = {}".format(k))
    print("Acuracia do treino: {:.3}".format(train_acc))
    print("Acuracia do teste: {:.3}".format(val_acc))
    return [k, train_acc, val_acc];

O parmetro k informa a quantidade de vizinhos. Na fase de classificao, k uma constante definida pelo usurio e um vetor no rotulado classificado atribuindo-se o rtulo que mais frequente entre as k amostras de treinamento mais prximas a esse ponto de consulta.

Random Forest¶

Floresta Aleatria um algoritmo de aprendizagem supervisionada que cria uma floresta de um modo aleatrio. A floresta criada uma combinao (ensemble) de rvores de deciso, em vrios casos treinados com o mtodo de bagging.

In [31]:
#
# Funcao que treina o modelo e mostra os resultados utilizando o Random Forest
#

def test_random_forest(N):
    clf = ensemble.RandomForestClassifier(n_estimators=N)

    clf = clf.fit(x_train, y_train)

    train_acc = clf.score(x_train, y_train)
    val_acc = clf.score(x_test, y_test)

    print("N = {}".format(N))
    print("Acuracia do treino: {:.3}".format(train_acc))
    print("Acuracia do teste: {:.3}".format(val_acc))
    return [N, train_acc, val_acc];

O parmetro N define a quantidade de rvores que sero criadas pelo Random Forest. O hiperparmetro N que indica o nmero de rvores construdas pelo algoritmo antes de tomar uma votao ou fazer uma mdia de predies. Em geral, uma quantidade elevada de rvores aumenta a performance e torna as predies mais estveis, mas tambm torna a computao mais lenta.

Gradient Tree Boosting¶

Gradient Tree Boosting uma tcnica de aprendizado de mquina para problemas de regresso e classificao, que produz um modelo de previso na forma de um conjunto de modelos de previso fracos, a partir de rvores de deciso. Ele constri o modelo em um modo de estgio, como outros mtodos de otimizao, e os generaliza ao permitir a otimizao de uma funo de perda arbitrariamente diferencivel.

In [38]:

def test_gradient_tree_boosting(N):
    clf = GradientBoostingClassifier(n_estimators=N)

    clf = clf.fit(x_train, y_train)

    train_acc = clf.score(x_train, y_train)
    val_acc = clf.score(x_test, y_test)

    print("Acuracia do treino: {:.3}".format(train_acc))
    print("Acuracia do teste: {:.3}".format(val_acc))
    return [N, train_acc, val_acc];

Avaliação dos métodos¶

Para avaliar os mtodos foi escrita a funo evaluation_method que recebe o classificador a ser avaliado. Todos os mtodos contaram com os melhores valores encontrados na etapa anterior. A avaliao contou com a acurcia e a validao cruzada k-fold com k igual a 5. A funo abaixo exibe os valores da acurcia, a curva de ROC a partir da validao cruzada k-fold, a preciso e revocao dos mtodos. Alm disso a funo salva os valores em uma lista para que no final os valores encontrados sejam comparados.

def evaluation_method(clf, nameMethod):
    # Validao cruzada utilizando k-fold
    cv = KFold(n_splits=5)

    # Define variveis 
    tprs = []
    aucs = []
    mean_fpr = np.linspace(0, 1, 100)
    i = 0
    plt.figure(figsize=(8, 8))

    # Calcula as probabilidades para cada conjunto definido pelo k-fold
    for train, test in cv.split(x, y):
        clff = clf.fit(x[train], y[train])
        y_pred = clff.predict(x[test])
        y_prob = clff.predict_proba(x[test])[:, 1]
                
        # Computa a curva ROC e a rea de curvatura
        fpr, tpr, thresholds = roc_curve(y[test], y_prob)

        tprs.append(interp(mean_fpr, fpr, tpr))
        tprs[-1][0] = 0.0
        roc_auc = auc(fpr, tpr)
        aucs.append(roc_auc)
        plt.plot(fpr, tpr, lw=1, alpha=0.3,
                 label='ROC fold %d (AUC = %0.2f)' % (i, roc_auc))
        i += 1
    
    y_pred = clff.predict(x_test)
    precisionMethod = precision_score(y_test, y_pred)
    recallMethod = recall_score(y_test, y_pred)

    print("")
    print("\tPrecision: %1.3f" % precisionMethod)
    print("\tRecall: %1.3f" % recallMethod)
    
    namesMethods.append(nameMethod)
    valuesMethods.append([precisionMethod, recallMethod])
    matrixMethods.append([nameMethod, precisionMethod, recallMethod])

    plt.plot([0, 1], [0, 1], linestyle='--', lw=2, color='r',
             label='Chance', alpha=.8)

    mean_tpr = np.mean(tprs, axis=0)
    mean_tpr[-1] = 1.0
    mean_auc = auc(mean_fpr, mean_tpr)
    std_auc = np.std(aucs)
    plt.plot(mean_fpr, mean_tpr, color='b',
             label=r'Mean ROC (AUC = %0.2f $\pm$ %0.2f)' % (mean_auc, std_auc),
             lw=2, alpha=.8)

    std_tpr = np.std(tprs, axis=0)
    tprs_upper = np.minimum(mean_tpr + std_tpr, 1)
    tprs_lower = np.maximum(mean_tpr - std_tpr, 0)
    plt.fill_between(mean_fpr, tprs_lower, tprs_upper, color='grey', alpha=.2,
                     label=r'$\pm$ 1 std. dev.')

    plt.xlim([-0.05, 1.05])
    plt.ylim([-0.05, 1.05])
    plt.xlabel('False Positive Rate')
    plt.ylabel('True Positive Rate')
    plt.title('Curva ROC (' + nameMethod + ')')
    plt.legend(loc="lower right")
    plt.show()
